# Chunksmith — Forge (1.21.x)

[![Discord](https://img.shields.io/badge/Discord-join-5865F2?logo=discord&logoColor=white)](https://discord.gg/2ZxzbCzAHe)

Forge builds of Chunksmith for Minecraft 1.21–1.21.11. One self-contained build per Minecraft version:

- [`1.20.6`](https://github.com/Kishku7/Chunksmith/tree/1.21.x/Forge/1.20.6) — Minecraft 1.20.6
- [`1.21.1`](https://github.com/Kishku7/Chunksmith/tree/1.21.x/Forge/1.21.1) — Minecraft 1.21.1
- [`1.21.4`](https://github.com/Kishku7/Chunksmith/tree/1.21.x/Forge/1.21.4) — Minecraft 1.21.4
- [`1.21.8`](https://github.com/Kishku7/Chunksmith/tree/1.21.x/Forge/1.21.8) — Minecraft 1.21.8

## Not built here

Minecraft 1.21.9–1.21.11 are **not** built for Forge: ForgeGradle 6 cannot build them and there is no FG7. Use NeoForge for 1.21.9+.
